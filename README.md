# AALI OS

An interactive operating system-inspired portfolio built with Next.js, TypeScript, and modern web technologies.

> Booting soon...

## Vision

Traditional portfolio websites are boring.

AALI OS reimagines a developer portfolio as an interactive operating system where visitors explore projects, experience, and achievements through a real terminal interface.
